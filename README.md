\# Solomon Odur Research Technologies (SORT) Website



This repository contains the official static website for Solomon Odur Research Technologies (SORT).



SORT is a Uganda-based research, data analytics, monitoring and evaluation, GIS, laboratory systems, and innovation consultancy brand focused on transforming evidence into practical solutions for Africa and beyond.
